# Shkolo
Basic platform for bulgarian school. Teachers, students and administration can use it to ease their workflow and do their job remotely.

The whole project is written on C# and sql queries! It's a windows from application.

-> DATABASES are made with MySQL database management system.

1.Tables:

students - store student profiles
teacher - store teacher profiles
administartion - store administation profiles
classes - store different classes
grades - store all grades of all students
grade_types - represent grade of which subject a student receive (id-subject relation)
events - store information about all created events by the administrations
event_invites - store invites for events
The code is in 'source code' folder!

If you want to install the app, go in 'EXE' folder and click 'Shkolo'.

THE PLATFORM IS CODED TO WORK FOR ONLY ONE SCHOOL!
